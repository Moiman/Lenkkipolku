# Lenkkipolku app

Features:
- Ability draw paths on map and measure their distance.
- Save paths and view and edit them later.

# Prerequisites
Node, npm and docker

# Run app
To run app:
1. Create .env based on .env.example
2. docker compose up
3. Go to http://localhost:$FRONTEND_PORT ( http://localhost:4173/ in .env.example)
